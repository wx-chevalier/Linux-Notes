# Nano

Nano 是 Linux 上一个简单的文本编辑器，比起 Vim 来说它轻量很多，易于上手。

| 含义     | 快捷键             |
| -------- | ------------------ |
| 标记     | `Ctrl+6` / `Alt+A` |
| 复制整行 | `Alt+6`            |
| 剪贴整行 | `Ctrl+K`           |
| 粘贴     | `Ctrl+U`           |
| 查找     | `Ctrl+W` (WhereIs) |
| 继续查找 | `Alt+W`            |
| 上一页   | `Ctrl+Y`           |
| 下一页   | `Ctrl+V`           |
| 保存     | `Ctrl+O`           |
| 退出     | `Ctrl+X`           |
